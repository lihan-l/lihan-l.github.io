ROB102 - Project 1: Wall Following

From Drive Square to Wall-Follower --- Team4: Julius & Lihan

The first part of this project is about controlling the robot to drive a square three times. Since there will be some offset between the actual moving direction and the desired moving direction, we manually adjust the "wz" term in the "drive" function for each turning. Putting thses code inside a for loop, we get the motion as shown in the video below.

In the second part of the project, we first get the direction of velocity that is parallel and perpendicular to the wall, then use bang-bang control to get the proper magnitude of velocity. This strategy does not have the risk of overshoot like p-control, but suffers from unsmooth motion. We can combine both strategy to behave both smoothly and stably.
